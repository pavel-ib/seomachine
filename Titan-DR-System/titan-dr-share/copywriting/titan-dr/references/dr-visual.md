# TITAN DIRECT RESPONSE — VISUAL COMPANION NOTES

### 6B. VISUAL COMPANION NOTES

**The 88ms Rule:** Your visual must register as novel within 88 milliseconds to trigger amygdala engagement before conscious processing. If it looks like every other ad, it's invisible.

**Format Defaults:**
- Feed: 4:5 vertical (1080x1350) — 15% better performance than square
- Stories/Reels: 9:16 (1080x1920) — design for stricter Reels margins
- Captions: MANDATORY. 85% watch without sound. 80% more likely to watch to completion with captions.

**Visual Principles:**
- Faces: 11x more likely to get noticed. Use gaze direction toward CTA.
- Dark backgrounds outperform light by 131% in clicks (Facebook = white background, light ads blend in)
- Text on image: 3-4 words max. Left-aligned outperforms centered.
- UGC aesthetic outperforms polished studio by 25% lower CPA / 37% greater engagement
- Message match: Ad visual → landing page visual must be consistent. Mismatch = 200% conversion loss.

**Creative Fatigue:**
- Refresh every 7-10 days (top brands average 10.4 days)
- Frequency above 2.5 = performance decline begins
- Always have 3 pipeline stages: Live (3-5 active) → Ready (5-10 waiting) → Production (10+ in development)

**Video Structure Quick Reference:**

| Length | Structure |
|--------|-----------|
| 15s | Hook (3s) → Value (5s) → Solution (4s) → CTA (3s) |
| 30s | Hook (5s) → Problem (5s) → Solution (10s) → Proof (5s) → CTA (5s) |
| 60s | Hook (3s) → Setup (7s) → Agitate (20s) → Solution (20s) → Re-hook + Proof (15s) → CTA (5s) |
| 90s+ Mini-VSL | Hook (5s) → Qualify (10s) → Educate (30s) → Soft Pitch (15s) → Re-hook (15s) → Objections (15s) → Hard Pitch (15s) → Risk Reversal (15s) → CTA (15s) |

**Hook Rate Benchmarks:**
- Above 35%: Excellent — scale and create variations
- 25-35%: Decent — test alternatives
- Below 25%: Kill it — new approach needed

## SURVIVEX VISUAL DIRECTION

**Product Photography Priorities:**
1. **Open Kit Hero Shot** — Kit fully open showing all 4 color-coded sections (red, blue, teal, black) with labels visible. This is the #1 conversion image. The organization IS the sell.
2. **Component Close-ups** — Zip Stitch wound closure strips, Israeli bandages, trauma shears, burn treatment. Show individual premium components to justify the price point.
3. **In-Context Usage** — Person calmly treating a (non-graphic) injury with the kit open beside them. Settings: kitchen, campsite, car trunk, soccer field sideline.
4. **Size/Scale Reference** — Kit next to common objects (backpack, car console) to show portability. MOLLE attachment to bag/vest.
5. **Color System Detail** — Close-up of color-coded compartment labels. Make the organizational system the hero.
6. **Family Context** — Parent showing child the kit, family loading kit into car for a trip. Warm, capable, not paranoid.

**Color Psychology (SurviveX System):**
- Red sections = urgency, wound care, immediate attention
- Blue sections = clean, antiseptic, medical trust
- Teal sections = hygiene, freshness, care
- Black sections = tools, precision, strength
- Overall brand feel: Calm confidence, professional organization. NOT tactical/military. NOT clinical/sterile. Think "capable family" not "doomsday prepper."

**Video Content Priorities:**
1. **Unboxing + First Impression** — UGC creator opens kit, reacts to organization. "I've never seen a first aid kit like this."
2. **Speed Test** — "Find wound closure strips in this generic kit vs. SurviveX." Side-by-side timed comparison.
3. **Zip Stitch Demo** — Demonstrate wound closure strip application on a practice surface (NOT on actual wound in ad creative).
4. **Real Scenario Walkthrough** — "Your kid falls and scrapes their knee. Here's exactly what you do." Walk through the color-coded system.
5. **EMT Endorsement** — Chase Carter or other EMT reviewing the kit, explaining why the organization mirrors professional trauma bags.
6. **Family Preparedness** — Family loading kit into car, parent showing teenager the system, calm organized response drill.

## STATIC IMAGE AD PRODUCTION CHECKLIST (5 Phases)

### Phase 1: Briefing and Concept
- Define campaign objective (sales, signups, awareness)
- Identify target audience and pain points
- Determine key message and emotional hook
- Establish brand guidelines (colors, fonts, logo placement)
- Select archetype from the 10 frameworks
- Gather reference examples from swipe file

### Phase 2: Asset Collection
- Source/shoot product images
- Gather lifestyle imagery or stock photos (authentic, not stocky)
- Collect brand assets (logos, fonts, color codes)
- Review existing high-performing creatives

### Phase 3: Design and Creation
- Create hook that captures attention in 0.3 seconds
- Maintain visual hierarchy (headline > image > CTA)
- Keep design uncluttered (one message per visual)
- Ensure CTA is clear and stands out
- Apply safe zone template
- Create 3+ variations for A/B testing

### Phase 4: QA Review
- Verify logo positioning and sizing
- Check brand colors are correct
- Confirm legal disclaimers included (if required)
- Validate CTA clarity and visibility
- Test resize integrity across formats (1:1, 4:5, 9:16)
- Proofread all copy
- Run 3-second comprehension test

### Phase 5: Export and Delivery
- Export in required formats (JPEG, PNG, WebP)
- Create platform-specific sizes
- Compress files for optimal load time
- Name files using naming convention
- Archive source files

## VIDEO AD PRODUCTION CHECKLIST

### Pre-Production
- Define objective and target audience
- Select video structure template (15s, 30s, 60s, or Mini-VSL)
- Write script with clear hook, problem, solution, proof, CTA
- Create storyboard or shot list
- Plan for multiple aspect ratios (9:16, 1:1, 16:9)
- Organize talent, locations, and equipment
- Brief talent on TikTok Crop framing and eye contact

### Production
- Frame shots for multiple aspect ratios
- Capture b-roll and alternate takes
- Ensure audio quality (use external mic when possible)
- Get 5+ hook variations for testing
- Film with sound-off viewing in mind

### Post-Production
- Edit with retention editing principles (cut every 1.5-4 seconds for short-form)
- Add captions (mandatory for 85% sound-off viewing)
- Include sound design and music
- Apply color correction
- Add text overlays and graphics
- Insert re-hook at 45-60 second mark for longer videos
- Create clear CTA (visual and verbal)

### Export Settings
- Export high-quality master file (ProRes 422 HQ or similar)
- Platform-specific exports:
  - TikTok: 9:16, 21-34 seconds optimal
  - Instagram Reels: 9:16, 21-60 seconds
  - Facebook Feed: 4:5 or 1:1
- Use MP4 with H.264 codec for compatibility
- Create caption files (SRT) if needed

## UGC CREATOR BRIEF TEMPLATE (9 Sections)

### Section 1: Campaign Objective
- Single clear goal (Add to Cart, Free Trial, App Install)
- Define what winning looks like (target metrics)

### Section 2: Brand Overview
- Mission and values — SurviveX: "Prepared. Organized. Life-saving." Organization Under Pressure positioning.
- What makes product unique — color-coded labeled compartments (red=wounds, blue=antiseptic, teal=hygiene, black=tools), EMT-P designed by Chase Carter, Zip Stitch wound closure technology
- Brand personality — Calm, confident, direct. Like a friend who happens to be an EMT. Not tactical-bro, not fear-mongering.

### Section 3: Target Audience
- "The Protector" — responsible everyday person, 80% male, 28-45, family provider. Wants to be capable and prepared without being a "prepper."
- Include 2-3 verbatim customer reviews for authentic voice

### Section 4: Key Message and Hook
- Product value proposition
- Emotional hook to grab attention
- 3-5 hook options for creator to paraphrase

### Section 5: Script Structure
- Cold Open (0-2s): Visual hook
- Problem (2-5s): Agitate the pain
- Demo/Result (5-12s): Show solution working
- Proof (8-12s): Social proof overlay
- CTA (12-15s): Clear next step

### Section 6: Deliverables and Specifications
- Length (15-60s typically)
- Aspect ratio (9:16 for vertical)
- Resolution (1080x1920)
- Number of variations required
- Caption needs

### Section 7: Visual References
- 2-3 examples of content you love
- Mood board or style guide

### Section 8: Do's and Don'ts
- Mandatory elements: Show open kit with color-coded compartments visible, demonstrate Zip Stitch or component retrieval, show MOLLE compatibility
- DO: Show kit in real-world settings (car, hiking pack, kitchen), show calm confident usage, show color-coded organization clearly
- DON'T: Use graphic injury imagery, show blood/gore in ad creative, use military/tactical-bro aesthetic that alienates mainstream families
- Prohibited elements: Competitor logos, graphic wound photos in ads, "saves lives" as direct product claim in ad creative

### Section 9: Compliance Notes
- FTC disclosure requirements for endorsements and testimonials
- Meta ad policies: no graphic injury imagery, no personal attributes violations, no medical outcome claims
- Medical disclaimer: "Not a substitute for professional medical care. Always call 911 for serious injuries."
- SurviveX color system for brand consistency: Red (wound care), Blue (antiseptic), Teal (hygiene), Black (tools)

## CREATIVE ASSET ORGANIZATION

### File Naming Convention
Format: `ClientName_ProjectType_MediaType_Date_Version`
Example: `BrandX_Summer2025_Video_9x16_20250128_v01.mp4`
- Use underscores, not spaces or special characters
- Include version numbers (v01, v02, v03)
- Use YYYYMMDD date format for chronological sorting

### Folder Structure
```
/Client Name
  /Campaign Name
    /01_Briefs
    /02_Source_Assets
      /Photos
      /Videos
      /Audio
      /Fonts
    /03_Working_Files
    /04_Exports
      /Static (by platform)
```

### Performance Tagging System
Tag every creative with:
- Hook type used
- Archetype
- Performance tier (Winner/Test/Failed)
- Date range active
- Key metrics (CTR, CPA, ROAS)

## LONG-FORM VIDEO CASE STUDIES (11-15)

### Case Study 11: Squatty Potty (Ecommerce)
- **Visual Concept:** 3-minute comedic video — unicorn pooping rainbow soft-serve while prince explains proper toilet posture. Produced by Harmon Brothers.
- **Results:** 600% increase online sales, 400% retail, 40M+ Facebook views, $38M sales. Investment: $200K production + $50K seeding = $250K.
- **Visual Psychology:** Humor transforms uncomfortable topic into shareable content. Educational component builds credibility.
- **Repeatable Principle:** For taboo products, use humor to overcome resistance. Long-form allows complete story arc and persuasion.

### Case Study 12: Lume Deodorant (Personal Care)
- **Visual Concept:** 4-minute musical number — caricature of French elegance. Original music created from scratch.
- **Results:** 7M views, 30K shares first month, 526% increase in sales, revenue growth $1.7M → $50M.
- **Visual Psychology:** Original music creates memorability. Entertainment value keeps viewers watching 4 minutes.
- **Repeatable Principle:** For sensitive issues, entertainment is your friend. Long-form works when genuinely entertaining.

### Case Study 13: Dr. Squatch (Personal Care)
- **Visual Concept:** 3-minute explainer starring comedian as Dr. Squatch character. Direct-to-camera with comedic delivery. "You are not a dish, you are a man."
- **Results:** 300M+ YouTube views, 100K+ shares, highest performing ad of 2020, revenue $5M → $100M in 24 months, acquired by Unilever for ~$1.5B.
- **Visual Psychology:** Character-driven memorability. Masculine messaging with humor. Educational content builds trust.
- **Repeatable Principle:** Create memorable character or persona. Combine education with entertainment.

### Case Study 14: ClickFunnels / Russell Brunson (Info Products)
- **Visual Concept:** 2-hour webinar presentation teaching Funnel Hacking strategy before $1,000 product pitch. Value-first, pitch-second structure.
- **Results:** 38% of first room purchased ($380K), scaled to 10K paying customers in first year, $100M+ annual revenue.
- **Visual Psychology:** Value delivery builds trust before asking for sale. Founder-led creates personal connection. Scarcity in webinar format.
- **Repeatable Principle:** For high-ticket info products, long-form educational content converts better than short ads.

### Case Study 15: Purple Mattress (Ecommerce)
- **Visual Concept:** Long-form video featuring Goldilocks and the Egg Test — demonstrating product superiority through memorable visual proof (raw eggs not breaking on mattress).
- **Results:** 4.1M YouTube views after one month, 38-49% view-through rate, helped Purple become $1.1B company.
- **Visual Psychology:** Visual proof demonstration creates believability. Humor makes ad memorable. USP clearly demonstrated.
- **Repeatable Principle:** For products with demonstrable differentiation, show do not tell. Create memorable visual tests viewers can describe to others.
