# TITAN WORDS — APPROVED POWER LIBRARY

> Words are the ammunition. This library is the armory.
> Every word earns its place or gets cut.

---

## CATEGORY 1: CURIOSITY & MYSTERY (Open Loops)

These words create information gaps the brain must resolve:

**Primary Curiosity Triggers:**
strange / unusual / odd / surprising / unexpected / hidden / secret / overlooked / unknown / forgotten / buried / discovered / reveals / uncovers / exposes / confesses / truth / real reason / what nobody tells you / what they don't want you to know / the one thing / the missing piece

**First Aid Curiosity Triggers:**
what EMTs actually carry / the supplies most kits are missing / what first responders know / the component nobody talks about / what happens in the first 5 minutes / the gap between injury and help / the one thing cheap kits never include

**Curiosity Phrase Starters:**
- "The strange reason why most first aid kits fail in real emergencies..."
- "Why most families never check what's actually inside their kit..."
- "The overlooked component that EMTs carry but no consumer kit includes..."
- "What your first aid kit is missing that could matter most..."
- "The real reason your 'complete' kit isn't complete..."
- "Here's what actually happens in the first 5 minutes of an emergency..."

## CATEGORY 2: URGENCY & SCARCITY

Create genuine time/availability pressure:

**Urgency Words:**
now / today / immediately / tonight / this week / before it's gone / while supplies last / limited batch / last chance / expires / deadline / running out / only X left / selling fast / almost gone

**First Aid Urgency Triggers:**
before the next emergency / before supplies expire / before you need it / when seconds count / when it matters most / before summer starts / before your next trip / before camping season

**Scarcity Phrase Templates:**
- "Only [number] kits from this production run — 900D polyester sourced from a single supplier"
- "Once this batch is gone, it's gone — we don't cut corners on materials"
- "We limit production to maintain Chase Carter's quality standards on every kit"
- "Free shipping over $75 ends [specific date]"

> **Rule:** Always give a reason for scarcity. Reason-based scarcity converts. Baseless scarcity triggers distrust.

## CATEGORY 7: SPECIFICITY AMPLIFIERS

Make any claim more credible instantly:

**Number Patterns That Signal Authenticity:**
- Use odd, non-round numbers: "247 components" not "250+" / "23.7%" not "24%"
- Add decimals to stats: "4.8 out of 5 stars" not "5 stars"
- Specific component counts: "250 expert-curated components" / "18 different wound care supplies"
- Specific money: "$120.99" not "around $120" / "$0.48 per component" not "affordable"
- Specific locations: "Designed in Virginia, USA" / "900D polyester from certified suppliers"
- Specific credentials: "Chase Carter, EMT-P, 10+ years field experience" not "designed by a professional"

**Specificity Phrase Templates:**
- "In a survey of [specific number] customers, [X]% reported feeling more prepared..."
- "The color-coded system uses [specific number] labeled compartments for [specific categories]..."
- "[Name], [age], [specific location]: '[testimony about a specific scenario]'"
- "At $[amount] per component — less than the cost of a single ER co-pay..."
- "200,000+ kits sold. 4,200+ five-star reviews. Designed by one EMT-P."

sharp / burning / throbbing / stinging / bleeding / raw / torn / swollen / heavy / panicking / shaking / fumbling / frantic / frozen / rushing / calm / steady / firm / smooth / organized / labeled / color-coded / sealed / sterile / secure / mounted / accessible / immediate

**Transformation Before/After Sensory:**

| Before State | After State |
|-------------|-------------|
| "Fumbling through a jumbled plastic box while someone is bleeding" | "Opening a color-coded kit and finding exactly what you need in seconds" |
| "That sick feeling when you realize you have nothing useful" | "The calm confidence of knowing you're equipped for this" |
