## Usage Guidelines

### When Writing New Content
1. Check which cluster the topic belongs to
2. Target the appropriate cluster or long-tail keyword as the primary target
3. Include 2–3 semantically related keywords in the article
4. Reference the pillar content/product for that cluster
5. Include LSI keywords naturally throughout
6. Link to related cluster articles (see internal-links-map.md)

### When Planning Content Calendar
1. Review the keyword opportunity pipeline (High → Medium → Low)
2. Align with seasonal keyword opportunities by quarter
3. Build out one cluster at a time before moving to the next
4. Prioritize "not yet created" content gaps in the high-volume tables
5. Track ranking changes monthly after publishing

### Keyword Placement Checklist
- [ ] Primary keyword in H1 title
- [ ] Primary keyword in meta title and meta description
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Secondary keywords in additional H2/H3 headings
- [ ] LSI keywords distributed naturally throughout
- [ ] Primary keyword in URL slug
- [ ] Primary keyword in image alt text (at least one image)
